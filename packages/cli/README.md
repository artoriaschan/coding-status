# cdps

Cloud provider usage statusline for Claude Code CLI

This is the public CLI package that re-exports `@cdps/core`. See the root README.md for full documentation.

## Installation

```bash
npm install -g cdps
```

## Usage

```bash
cdps init
cdps add
cdps statusline
```

See [GitHub repository](https://github.com/artoriaschan/coding-plans-statusline) for complete documentation.
